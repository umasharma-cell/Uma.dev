## Packages
framer-motion | Essential for complex animations and page transitions
react-icons | Comprehensive icon library for skill logos (SiReact, SiNodedotjs, etc.)
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes without conflicts

## Notes
- Dark mode is default and primary.
- Using 'Space Grotesk' for headings and 'Inter' for body.
- Animations should be smooth (spring physics) and stagger on load.
- Background should be deep, perhaps with a subtle animated gradient or particle effect.
